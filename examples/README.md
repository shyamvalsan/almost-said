# examples

`sky-demo.json` is a synthetic export in the same shape the visualizer expects: a short "Why is the sky blue?" chat with per-layer lens predictions on every token. It is generated by the page's built-in demo, so it is not a capture of any real model, but it exercises the full format (prompt and generated tokens, per-layer `top_tokens` / `top_probs`, a named lens `type`, and noise tokens that get filtered).

Drop it onto the page, or use the **watch the demo** link, which loads the same data from memory.
